# Casse du Casino Itinérant

## Description :
Mission de braquage d'un camion-casino mobile pour ESX Legacy. Neutralise les gardes, pirate le système, vole les jetons et repars avec du black money.

## Fichiers :
- `config.lua` : configuration des emplacements, récompenses, etc.
- `client.lua` : logique client
- `server.lua` : logique serveur
- `fxmanifest.lua` : déclaration du script

## Dépendances :
- ESX Legacy
- oxmysql
